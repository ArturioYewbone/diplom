# Game of Life

Реализация "Жизни" Конвея (Conway's Game of Life) на C++ с использованием SDL2 для графического отображения.

  Требования

- Компилятор C++ с поддержкой стандарта C++17 (например, g++)
- CMake версии 3.10 и выше
- Библиотека SDL2 (dev-пакет)
- Утилита pkg-config

  Установка зависимостей (Ubuntu)

```bash
sudo apt update
sudo apt install build-essential cmake libsdl2-dev pkg-config
```
  Сборка проекта
Перейдите в корневую папку проекта (где находятся CMakeLists.txt и main.cpp).
Создайте и откройте папку сборки:
```bash
mkdir build
cd build
cmake ..
make
```
После сборки появится исполняемый файл GameOfLife.

  Запуск
Запуск из папки build:
```bash
./GameOfLife [rows] [cols] [delayMs]
```
Параметры:
rows — количество строк на игровом поле (по умолчанию 80)
cols — количество столбцов (по умолчанию 80)
delayMs — задержка между обновлениями (мс, по умолчанию 100)

Пример:
```bash
./GameOfLife 50 100 75
```

  Примечания
Проект работает в оконном режиме с фиксированным размером клетки (10 пикселей).
При запуске можно задать только количество клеток и скорость, размеры окна рассчитываются автоматически.
